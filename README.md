# 自动点击器

一个基于 Python 的跨平台自动点击工具，支持录制屏幕点击位置并自动重复执行。

## 功能特性

- **录制点击位置** — 通过鼠标点击录制屏幕坐标，支持自定义最大录制数量
- **自动循环点击** — 按录制顺序自动点击，支持设置重复次数和间隔时间
- **编辑已录制位置** — 双击列表中的位置，3 秒倒计时后点击屏幕即可替换坐标
- **窗口始终置顶** — 操作时窗口不会被遮挡
- **快捷键停止** — 按 `ESC` 随时停止执行

## 环境要求

- Python 3.7+
- 依赖库：`pyautogui`、`pynput`

## 安装

```bash
pip install -r requirements.txt
```

## 运行

### Windows

```bash
python auto_clicker.py
```

### macOS

双击 `mac双击运行.command` 即可运行（首次需赋予执行权限：`chmod +x mac双击运行.command`）。

## 使用说明

1. 设置 **最大位置数**（默认 6）
2. 点击 **开始录制**，3 秒倒计时后在屏幕上点击需要自动点击的位置
3. 达到最大位置数后自动停止录制，也可手动点击 **停止录制**
4. 设置 **重复次数** 和 **间隔时间**
5. 点击 **开始执行**，程序将按顺序自动点击录制的位置
6. 按 `ESC` 或点击 **停止执行** 可随时中断

### 编辑已录制位置

双击列表中的某个位置 → 3 秒倒计时 → 点击屏幕新位置 → 坐标自动更新。

## Mac 权限设置步骤
### 方法一：授权终端
1. 打开 系统设置 → 隐私与安全性 → 辅助功能
2. 点击左下角 🔓 解锁（输入密码）
3. 点击 + 按钮
4. 添加 终端 ：
   - 路径： 应用程序 → 实用工具 → 终端.app
5. 勾选终端的复选框
6. 重启终端 ，再运行程序

### 方法二：授权 Python（推荐）
1. 打开 系统设置 → 隐私与安全性 → 辅助功能
2. 点击 + 按钮
3. 按 Cmd + Shift + G 输入路径：
/Library/Frameworks/Python.framework/Versions/3.12/bin/
- 选择 python3 文件添加
- 勾选复选框
- 重启程序
